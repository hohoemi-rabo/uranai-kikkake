プライバシーポリシー

本プライバシーポリシーは、モバイルアプリ「占いキッカケ」(以下「本アプリ」)の利用に関し、開発・運営者である ほほ笑みラボ(以下「当方」)が取得・利用する情報の取り扱いについて定めるものです。

1. 取得する情報

  • 認証 ID: Google / Apple ID から取得される識別子(sub)
  • 利用回数: 1 日 3 回の利用制限を管理するためのカウント
  • 撮影画像: 診断のためにアップロードされた画像(処理直後に破棄)

2. 利用目的

  • AI による顔相・手相・相性診断の実行
  • 1 日 3 回の利用制限の管理
  • 不正利用の防止

3. 第三者提供

撮影画像は診断のため、Google の Gemini API に送信されます。Gemini における画像の取り扱いについては、Google のプライバシーポリシーをご参照ください。

認証 ID は Google / Apple のログイン認証時にのみ利用され、それ以外の第三者には提供しません。

4. 保存期間

  • 認証 ID と利用回数: Cloudflare KV に 最大 48 時間 のみ保存され、日跨ぎ後に自動削除されます。
  • 撮影画像: 当方のサーバには一切保存しません。Gemini API への送信完了後、ただちに破棄します。

5. ユーザーの権利

アカウントの削除依頼や、個人情報に関するお問い合わせは下記までご連絡ください。

連絡先: rabo.hohoemi@gmail.com

6. 安全管理

取得した情報は暗号化された通信(HTTPS)で送受信され、Cloudflare Workers および Cloudflare KV のセキュリティ機構によって保護されます。

7. ポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更があった場合はアプリ内で告知します。
