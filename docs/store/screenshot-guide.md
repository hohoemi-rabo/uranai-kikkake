# スクリーンショット撮影ガイド

Play Store 提出用のスクリーンショット撮影手順。**チケット 25(Google ログイン)完了後** の作業。

---

## Play Store の要件

| 項目 | 要件 |
|------|------|
| 必要枚数 | **最低 2 枚、推奨 4〜8 枚** |
| 解像度 | **横 1080px 以上**(短辺 320px〜長辺 3840px) |
| アスペクト比 | 縦長(9:16)推奨 |
| 形式 | PNG または JPG |
| 容量 | 1 枚あたり 8MB 以下 |

参考: https://support.google.com/googleplay/android-developer/answer/9866151

---

## 撮影前提

- [ ] チケット 25(Google ログイン)完了済み
- [ ] `eas build --profile preview --platform android` 完了で実機 APK インストール済み
  - または `eas build --profile production` でも可
- [ ] 実機(Android 8+)or Android Studio エミュレータ(Pixel 6 / Pixel 7 推奨、1080×2400 程度)
- [ ] アプリの動作が正常で「会話のきっかけ」感ある画面が撮れる状態

---

## 撮影シーン推奨(6〜8 枚)

| # | 画面 | 訴求ポイント | コツ |
|---|------|------------|------|
| 01 | **ホーム画面**(charm 選択中) | 3 つのモード、シンプル UI | usage badge「今日の残り: 3 回」表示 |
| 02 | **タブ切替後**(palm or match) | 配色変化、モード別 | アクセント色が変わる様子 |
| 03 | **カメラ画面** | モード別フレーム | charm のフレーム + シャッターボタン |
| 04 | **診断中オーバーレイ** | わくわく感の演出 | モード絵文字 + パルス + メッセージ |
| 05 | **結果画面 上部** | タイトル + 動物 + メッセージ | 大きな文字、温かいトーン |
| 06 | **結果画面 下部** | 開運アイテム + アドバイス + おすすめの話題 | カラフルなカード並び |
| 07 | **保存した結果画像** | カメラロール保存機能 | 1080×自動高の縦長サンプル |
| 08 | **設定画面**(任意) | ログアウト・お問い合わせ・利用規約 | シニア向けのシンプルな構成 |

## キャプチャ方法

### Android 実機

- 電源 + 音量下ボタン **同時押し** → ギャラリーに保存
- ファイル名は `Screenshot_YYYYMMDD-HHMMSS.png`

### Android Studio エミュレータ

- 右側ツールバー → **カメラアイコン**(Take screenshot)
- デフォルト保存先: `~/Pictures/` 等

### Pixel デバイスでステータスバーを綺麗に

`adb shell settings put global sysui_demo_allowed 1` でデモモード有効化、時刻を 09:00、バッテリーを 100% にすると審査画像が綺麗。

## 編集(任意)

ストア審査では非加工でも OK ですが、より惹きつけるなら:

- **Figma / Canva** で説明テキストを重ねる
  - 例: 「💡 タップで魅力を発見」「✨ 家族との会話のきっかけに」
- **背景色や枠を統一**(各画像にブランドカラーの細帯を入れる)
- **文字は大きく**(シニア層が直感的に分かる訴求)

ただし「実際の画面と乖離する」加工は審査で差し戻されるので、説明テキスト追加程度に留める。
