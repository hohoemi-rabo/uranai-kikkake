# IARC コンテンツレーティング 質問票 回答ドラフト

Play Console の **アプリのコンテンツ → コンテンツのレーティング** に入力。
IARC(International Age Rating Coalition)の質問票に従って回答。

---

## カテゴリ選択

→ **その他のアプリ(エンターテイメント等)**

---

## 暴力

| 項目 | 回答 |
|------|------|
| 暴力的な表現はありますか? | **いいえ** |
| ファンタジー / 漫画的な暴力 | **いいえ** |
| 武器の使用 | **いいえ** |
| 流血表現 | **いいえ** |

---

## 性的表現

| 項目 | 回答 |
|------|------|
| 性的なコンテンツ | **いいえ** |
| ヌード / 露出 | **いいえ** |
| 性的な言葉遣い | **いいえ** |

---

## 言葉遣い

| 項目 | 回答 |
|------|------|
| 攻撃的 / 侮辱的な言葉 | **いいえ** |
| 不適切なユーモア | **いいえ**(ユーモアはあるが健全な範囲) |

---

## 薬物・アルコール・タバコ

| 項目 | 回答 |
|------|------|
| 薬物使用の描写 | **いいえ** |
| アルコール使用の描写 | **いいえ** |
| タバコ使用の描写 | **いいえ** |

---

## ギャンブル

| 項目 | 回答 |
|------|------|
| 実際のお金を賭ける | **いいえ** |
| シミュレーションギャンブル | **いいえ**(占いは結果を保証しないエンタメ、賭けの要素なし) |

---

## ホラー / 恐怖

| 項目 | 回答 |
|------|------|
| 恐怖を煽る内容 | **いいえ**(プロンプトで否定的・脅迫的な内容を禁止済み) |

---

## ユーザー間のやりとり

| 項目 | 回答 |
|------|------|
| ユーザー生成コンテンツの共有 | **いいえ** |
| 他ユーザーとのチャット / 通信 | **いいえ** |
| 位置情報の共有 | **いいえ** |
| 個人情報の共有 | **いいえ** |

---

## その他のセンシティブな内容

| 項目 | 回答 |
|------|------|
| 連絡先情報の収集(ユーザー入力) | **いいえ**(認証 ID のみ) |
| 位置情報の取得 | **いいえ** |
| デジタル購入 / アプリ内課金 | **いいえ**(無料) |
| 広告表示 | **いいえ**(広告なし) |

---

## 想定結果

- **全年齢 / 3+**(IARC ジェネリックレーティング)
- 日本国内: **CERO A**(全年齢相当)
- 米国: **ESRB Everyone**
- 欧州: **PEGI 3**

シニア層を中心に、家族で安心して楽しめる内容です。
